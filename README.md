# HirakeDoor

教室ドアの自動化プロジェクト

## 概要

教室にある横開きの扉を、センサーやモーターなどを用いて、自動ドア化するプロジェクトです。

## 動機

冬の教室で扉のすぐ横の席は、扉を開きっぱなしにされると非常に寒いです。
だからといって毎回自分で閉めたり、開けられるたびに閉めるようお願いするのも手間がかかります。
そこで始まったのがこのプロジェクトです。
第一段階として、扉におもりを取り付け、開けたら勝手に閉まるようなドアクローザー機構を作成しました。
概ね好評だったものの、開けるのが重くなるなどの問題もあり、第二段階としてセンサーやモーターなどを用いた自動ドア化を行いました。
